StockPred – Multiseries TFT Forecasting

A full research-grade time-series forecasting pipeline built with the Temporal Fusion Transformer (TFT) to predict ASX stock prices across 50+ companies. Includes feature engineering on OHLCV, macro indices, commodities, FX, technical indicators, calendar effects and news sentiment. Built with PyTorch Lightning, PyTorch Forecasting and YAML-driven configurations, with hyperparameter search and evaluation modules.

detect handwritten digits

A Neural Network project that can detect handwritten numbers. The dataset was split with a 6:1 ratio for train and test data. Uses adam optimizer for stochastic gradient descent and uses sparse categorical crossentropy as cost function

ai survellance

An integrated design project in college final year. The project is a platform in Django framework to stream video surveillance from recording devices, capable of detecting humans and setting off alarms in case of trespassing. Image preprocessing is done using OpenCV and trained with a OpenCV deep neural network module. The dataset was split with a 60/40 ratio for train and test data. Graded A+ by HOD Brigadier General Razzak and forwarded to military R&D for further research

rag pipeline

A research project on RAG pipeline using open source encoders and foundation llm. Used langchain with sentence transformer and data chunking: bge-base-en-v1.5 for embeddings in the vectorstore in this case a Chromadb store for ingestion, and used zephyr-7b-alpha from HuggingFace as the foundational model to feed the retrieved query and context in system prompt format for text generation
